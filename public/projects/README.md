Place your project thumbnail images here, named to match the repository slugs used in `src/App.jsx`.

Examples (filename -> repo slug):
- Al-Furqan.png                 -> Al-Furqan
- Fitjorn-Flutter.png           -> Fitjorn-Flutter
- Quotify-Flutter.png           -> Quotify-Flutter
- Quizzora-Flutter.png          -> Quizzora-Flutter
- Grocery-app-project.png       -> Grocery-app-project
- periodic-table-html-css.png   -> periodic-table-html-css
- flutter-covid-tracker-app.png -> flutter-covid-tracker-app
- weather-app-flutter.png       -> weather-app-flutter

Notes:
- Supported extensions: `.png`, `.jpg`, `.jpeg`, `.webp`.
- If a local image is missing, the component will fall back to the GitHub opengraph image.

Quick usage:
1. Put all attached images in this folder.
2. Ensure filenames contain the repo slug (exact or case-insensitive).
3. Start the dev server: `npm run dev`.

Helper script is available at `scripts/place-project-images.js` to assist with bulk renaming/copying.
